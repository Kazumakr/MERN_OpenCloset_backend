## Installation

1. Clone the repo
   ```sh
   git clone https://github.com/Kazumakr/
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Enter your MONGO_URL in `.env`. You can get it from here(https://cloud.mongodb.com)
   ```shell
   MONGO_URL = "ENTER YOUR CONNECTION STRING";
   ```
4. start
   ```sh
   npm start
   ```
